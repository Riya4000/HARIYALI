# hariyali_app

A new Flutter project.

## Getting Started

This project is a starting point for a Flutter application.
Hariyali is a smart greenhouse 🌱 IoT mobile application built with Flutter and Firebase that empowers small‑scale farmers to embrace sustainable agriculture with modern technology. By combining real‑time sensor monitoring, intuitive data visualization, voice assistant features, crop recommendations, and remote device control, Hariyali transforms traditional farming into a connected, efficient, and eco‑friendly experience. Designed to be simple yet powerful, the app helps farmers save time, reduce manual effort, and make smarter decisions for healthier crops and greener futures.
